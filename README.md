# Camino Boys Trip — shareable page

This is a standalone page for the group, kept separate from the family Travel Agent tree. `index.html` is the itinerary (day-by-day plan, route map, what's confirmed, what's open). `robots.txt` keeps it out of search engines. Repo: https://github.com/sigaramendrum/camino — live at https://sigaramendrum.github.io/camino/ once Pages is enabled.

To update: edit `index.html`, then `git add index.html && git commit -m "update" && git push`.
